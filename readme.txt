# SCM Assignment 3 — World Clock

A single-page website that displays live time across six global timezones.

## Files

| File        | Description                          |
|-------------|--------------------------------------|
| index.html  | Main page structure and JS clock logic |
| style.css   | All styling and layout               |
| README.md   | This file                            |

## Cities Covered

- New York (UTC −5)
- London (UTC +0)
- Lagos (UTC +1)
- Dubai (UTC +4)
- Tokyo (UTC +9)
- Sydney (UTC +11)

## How to Run

Open `index.html` in any modern browser. No build step required.
